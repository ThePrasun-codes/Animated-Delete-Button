🗑️ Animated Delete Button

A simple and smooth animated delete button built using HTML, CSS, and JavaScript. Clicking the button plays a trash can animation and automatically resets so it can be used again.

✨ Features

- Smooth delete animation
- Pure HTML, CSS, and JavaScript
- No external libraries required
- Lightweight and easy to customize
- Responsive design
